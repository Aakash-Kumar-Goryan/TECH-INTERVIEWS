# Design Case Studies

## URL Shortener
- Unique ID generation, redirection, analytics

## Distributed Cache
- Consistency, eviction, replication (e.g., Redis, Memcached)

## Messaging System
- Real-time delivery, ordering, durability (e.g., WhatsApp, Slack)

## Event Streaming Platform
- Partitioning, consumer groups, durability (e.g., Apache Kafka, Kinesis)

## File Storage/Sharing
- Chunking, deduplication, sharing (e.g., Dropbox, Google Drive)

## News Feed/Timeline
- Ranking, fan-out, real-time updates (e.g., Facebook, Twitter)

## Real-Time Chat
- WebSockets, message persistence, scaling (e.g., Messenger, Slack)

## Video Streaming
- Encoding, CDN, adaptive bitrate (e.g., YouTube, Netflix)

## Search Engine
- Crawling, indexing, ranking (e.g., Google Search)

## E-commerce Backend
- Inventory, payments, order management (e.g., Amazon)

## Ride Sharing
- Matching, pricing, real-time tracking (e.g., Uber, Lyft)
