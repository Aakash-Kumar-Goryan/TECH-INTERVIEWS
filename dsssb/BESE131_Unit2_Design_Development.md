# Unit 5: Designing Self-Learning Materials (SLMs)

## 5.1 What are SLMs?

**Self-Learning Materials** = Instructional materials enabling independent learning without teacher supervision.

### Characteristics of SLMs

| Feature | Description |
|---------|-------------|
| Self-explanatory | Clear without external help |
| Self-contained | All info included |
| Self-directed | Learner controls pace |
| Self-motivating | Engaging content |
| Self-evaluating | Built-in assessments |

---

## 5.2 Key Instructional Design Models

### ADDIE Model (Most Important)

```
ANALYZE ‚Üí DESIGN ‚Üí DEVELOP ‚Üí IMPLEMENT ‚Üí EVALUATE
```

| Phase | Activities |
|-------|------------|
| **Analyze** | Learner needs, goals, constraints |
| **Design** | Objectives, strategies, assessment plan |
| **Develop** | Create content, materials, media |
| **Implement** | Deliver instruction |
| **Evaluate** | Formative & summative assessment |

### Gagn√©'s 9 Events of Instruction

1. Gain Attention
2. Inform Objectives
3. Stimulate Recall
4. Present Content
5. Provide Guidance
6. Elicit Performance
7. Provide Feedback
8. Assess Performance
9. Enhance Retention

---

## 5.3 Universal Design for Learning (UDL)

| Principle | Focus |
|-----------|-------|
| Multiple Means of **Engagement** | WHY of learning |
| Multiple Means of **Representation** | WHAT of learning |
| Multiple Means of **Action/Expression** | HOW of learning |

## 7.1 Components of SLPM Unit
